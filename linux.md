#  Linux 🐧
## Introduction 
Linux is a free, open-source operating system modeled on Unix.From smartphones to cars, supercomputers and home appliances, 
home desktops to enterprise servers, the Linux operating system is everywhere. 

one of the most popular platforms on the planet, Android, is powered by the Linux operating system. 
An operating system is software that manages all of the hardware resources associated with your desktop or laptop.

## What is a “distribution?”
Linux has a number of different versions to suit any type of user. From new users to hard-core users, you’ll find a “flavor” of Linux to match your needs.  These versions are 
called distributions.

Nearly every distribution of Linux can be downloaded for free, burned onto disk (or USB thumb drive), and installed.

Each distribution has a different take on the desktop. Some opt for very modern user interfaces (such as GNOME and Elementary OS’s Pantheon), 
whereas others stick with a more traditional desktop environment (openSUSE uses KDE).

## Linux Commands
|Commands|Description|
|-----|-----|
|**View & Create**|
|ls| view/list all files in current location/path|
|ls -d| list only directory current location/path|
|ls -ltr| view/list all files with file permission,size,created |
|pwd|view current location path |
|cd /etc/..|nagivate inside directory|
|touch <file-name>| Crete a new file |
|mkdir <directory-name>|Create a new directory |
|sudo ln -s /etc/.../<source-file-name> /etc/.../<target-file-name>|Create a soft link between two file or folder in linux. If you used nginx, then it will be most useful command.eg, sudo ln -s /etc/nginx/sites-available/my-site /etc/nginx/sites-enabled/|
|cat <file-name>| View File|
|sudo nano <file-name>| Edit File|
|history|view previous commands |
|**Monitor/Process viewer**|
|df -h|view file system and used storage details|
|htop|view task manager in linux|
|**Service & Package Manager**|
|sudo systemctl start <service-name>|start the installed application service using cli. eg. sudo systemctl start nginx.service|
|sudo systemctl restart <service-name>|restart the running service|
|sudo systemctl stop <service-name>|stop the running service|
|sudo systemctl status <service-name>|View status of the service|
|sudo rm <file-name> or <directory-name> | remove file or directory |
|sudo rm -rf <file-name> or <directory-name> | Force fully remove file or directory |
|clear| clear terminal |
|sudo apt install <application-name>|Get application from linux package manager|
|sudo apt remove <application-name>| Remove installed application |
|sudo apt update | Update all installed application |
|sudo snap install <application-name>|Get application from snap store manager|
|sudo apt list --installed|Get list of packages installed |
